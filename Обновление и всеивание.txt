Обновление и всеивание
компонент(ы) - пересмотреть связи (и для запросов бд то же)
файл(ы) - если у арма нет соединений - не работает; читает форму, получает список компонентов
арм(ы) - лезет в проект, формирует список файлов
всё - лезет в конфиг со списком армов и формирует там их список

Обработка всего
	прочитали конфиг, получили перечень армов
	Если арма из перечня нет в базе - создать и запушить объекты ОРМ в список на обновление
	сформировали список армов на обновление готов
	Обновление_армов(список_армов)

Обновление_армов(список_армов)
	Перебор армов по списку
		Проверка доступности файла проекта
		Чтение файла проекта
			проверяем доступность каждого файла формы и дату последнего изменения
			файл недоступен - ошибка
			ищем наибольшую дату среди всех файлов - это дата обновления арма
			Для каждого файла проверить наличие в базе и дату обновления
			Если файла нет в базе - создать, поставить дату обновления
			список_форм для обновления готов
		Обновление_форм(список_форм)
		Проставить в базе дату обновления арма

Обновление_файлов(список_файлов)
	# Передаётся список объектов ОРМ, к какому арму относятся файлы уже известно
	Перебор списка файлов
		Разобрать файл
		Перебор компонентов
			Если соединение
				Достать соединение из базы по арму и имени
				Если нет - создать (приделать алиас формы спереди)
				Добавить объект в список
			Если не соединение
				Запросить из базы
				Если есть
					Контрольные суммы совпадают
						смотрим следующий
					не совпадают
						обновить текст запроса
				Если нет
					Посмотреть соединение
					Если нет в списке
						Запросить из базы
						Если нет в базе
							Создать (если имя без алиаса - приделать свой)
						Добавить объект в список
					Создать объект для компонента (имя, скуль, соединение, файл)
