
Список АРМов

Таблица с полями: "Название", "Дата обновления" с возможностью сортировки по ним, поиск по названию;
При выборе арма в таблице доступны следующие опции:
 - Редактор связей объекта;
 - Просмотр зависимостей (прямых, всех, уникальных)
 - Просмотр источников данных и изменяемых таблиц (вставка, удаление, обновление, вместе и в сочетаниях)
 - Пересчёт связей

Редактор связей
Показывает список связей, исходящих из рассматриваемого объекта
Поля таблицы: "Связанный объект", "Тип", "Комментарий", "Дата создания", "Подтверждено", "Сломано"
 - Связи можно редактировать и удалять; удаление и установка отметок "Подтверждено" и "Сломано" можно делать на несколько связей сразу;
  - При создании связи указывается связанный объект и тип связи, комментарий, дата заполняется автоматом, связь по умолчанию не сломана и подтверждена (так как создаётся руками);
  - При редактировании можно править всё, кроме даты и связанного объекта;

 Просмотр зависимостей объекта
 Список объектов с указанием типов, граф зависимостей

Работа со списком зависимых объектов
    Можно удалить связь с объектом
    Можно редактировать связь с объектом (коммент, подтвердить, сломать)
    Можно добавить связь с новым объектом
    Можно перейти к просмотру связей самого объекта
    Просмотр более подробных характеристик объекта (где-то сбоку, например)

 Список баз данных
 По отдельной базе можно посмотреть:
     Список зависимых АРМов
     Можно смотреть списки объектов БД (таблицы, функции скалярные и табличные, процедуры и представления), фильтрация списка
        Переход к зависимостям выбранного объекта
        Вывод более подробных характеристик объекта

 Полная карта зависимостей АРМов от баз (только армы и базы)

 Просмотр зависимостей объекта
    Арм - зависимости идут вниз
    Таблица - зависимости идут вверх
    Запросы армов - нужны ограничения
    Запросы в бд - зависимости идут и вниз, и вверх

Зависимости идут вверх, пока не упрутся в АРМ
Зависимости идут вниз, пока не упрутся в таблицу

Формы:
    Список баз;
    Список армов;
    Список объектов базы;
    Список завимостей объекта;
    Список связей объекта (выжимка предыдущего);
    Характеристики связи;
    Характеристики объекта:
        база
        арм
        таблица
        запрос
            из арма
            из базы

Список зависимостей объекта
    Можно удалить связь с объектом
    Можно редактировать связь с объектом (коммент, свойства, подтвердить, сломать)
    Можно добавить связь с новым объектом
    Можно перейти к просмотру связей зависимого объекта объекта
    Просмотр более подробных характеристик объекта (где-то сбоку, например)
    Режимы просмотра: 
        все зависимости; 
        прямые зависимости;
        уникальные зависимости;
        по операциям: запись, обновление, удаление, чтение (в сочетаниях);
            для запросов связь прямая (т.е. запрос читает из..., пишет в...)
            для таблиц связь обратная (т.е. в таблицу пишет..., из таблицы читает...)
        показ объектов определённого типа (напр., только таблицы)
